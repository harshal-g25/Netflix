# Media folder — add your files here

Place your photos, videos, and music in this folder. Paths match `anniversaryData.js` and `anniversaryConfig.js`.

## Required / recommended files

| File | Purpose |
|------|---------|
| `our-song.mp3` | Background music (floating player) |
| `hero-backdrop.jpg` | Home hero banner image |
| `hero-video.mp4` | Hero background video (optional, on Play) |

## Folders (create as needed)

- `memories/` — "Our Best Memories" row
- `moments/` — "Moments I'll Never Forget"
- `loving/` — "Continue Loving"
- `journey/` — Journey page slides and timeline photos
- `favorites/` — "Favorite Moments"
- `forever/` — "Forever Together"
- `letters/` — Love letter envelope images
- `future/` — Future dreams cards

Until you add files, the site shows styled placeholders with the exact path to use.
